    series.keltnerchannels

    A Keltner Channels indicator. If the type option is not specified, it is inherited fromchart.type.

    In TypeScript the type option must always be set.

    Configuration options for the series are given in three levels:

    1. Options for all series in a chart are defined in the plotOptions.series object.
    2. Options for all keltnerchannels series are defined in plotOptions.keltnerchannels.
    3. Options for one single series are given in the series instance array.
    Highcharts.chart('container', {
    plotOptions: {
        series: {
            // general options for all series
        },
        keltnerchannels: {
            // shared options for all keltnerchannels series
        }
    },
    series: [{
        // specific options for this series instance
        type: 'keltnerchannels'
    }]
});
